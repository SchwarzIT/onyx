# Infographics

A visual representations of data and information designed to make complex insights digestible.

## Usage

Graphical elements such as charts, diagrams, and maps, infographics help users identify patterns, trends, and outliers that might be difficult to discern in raw data.

## Implementation

For implementing high-quality charts and data visualizations, use Chart.js. To ensure these visualizations align with the design system, use the custom design system plugin for Chart.js. This plugin applies specific stylings to ensure visual consistency across all components.

---

With onyx, we do not try to re-invent the wheel for implementing charts so we recommend to use [Chart.js](https://www.chartjs.org) for infographics.

We provide a [custom Chart.js plugin](/development/packages/chartjs-plugin) to seamlessly provide onyx stylings to charts.
