## Enable support for older browser versions

If you must support specific older browser versions, use `lightningcss` with `@vitejs/plugin-legacy` or `nuxt-vite-legacy` (for Nuxt applications). These tools provide polyfills and syntax lowering.

The following Nuxt configuration example supports Chrome 120:

```ts
import { defineNuxtConfig } from "nuxt/config";

export default defineNuxtConfig({
  // Use `lightningcss` to support syntax lowering for modern CSS features
  vite: {
    build: {
      cssMinify: "lightningcss",
    },
    css: {
      lightningcss: {
        targets: {
          chrome: 120,
        },
      },
    },
  },

  // Use the `nuxt-vite-legacy` module to inject polyfills for relevant user agents
  legacy: {
    targets: ["chrome >= 120"],
  },
  modules: ["nuxt-vite-legacy"],
});
```
